Cotter

Example

To run the example project, clone the repo, and run pod install from the Example directory first.

Developer Notes

Follow this guideline and add your COTTER_API_KEY_ID and COTTER_API_SECRET_KEY under “Environment Variables”

Requirements

Installation

Cotter is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod 'Cotter'

Developer Notes:

Please run make init after you clone the project. Thanks!
